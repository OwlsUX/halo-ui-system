## Audit Coverage Requirements

Every page above should be checked for:

- shared customer-facing header
- shared customer-facing footer
- floating UI/workbench drawer availability
- consistent yellow promo bar behavior where customer chrome appears
- typography token usage and readable hierarchy
- spacing token usage and section rhythm
- desktop grid alignment at `1440x1000`
- tablet grid alignment at `1024x900`
- mobile layout at `390x844`
- no horizontal overflow
- no clipped text or CTA labels
- no broken local images, CSS, or JS
- no stale navigation links in the drawer
- no page-specific hardcoded values unless documented in a page audit
