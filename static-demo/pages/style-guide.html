Color

Color roles

Use semantic Halo roles first. Yellow is for purchase and promotional action. Blue is for links, selected states, and technical focus. Deep blue is the text and dark-surface foundation.

Action primary--halo-color-action-primary#FCD62D

Purchase CTAs, plan selection, promo action.

Link / selected--halo-color-link#2F93F3

Links, selected options, focus, technical accents.

Ink / dark--halo-blue-950#071826

Main text, dark bands, footer, dark buttons.

Page wash--halo-color-surface-page#F5F9FC

Default page background.

Raised surface--halo-color-surface-raised#FFFFFF

Cards, forms, account panels.

Product surface--halo-color-product-surface#F4F7F9

PDP galleries and product media wells.

Membership--halo-color-membership-surface#EDF7FF

Plans, Pack Perks, current account state.

Sale surface--halo-color-sale-surface#FFF3DF

Discount context and price support.

Success--halo-color-status-success-surface#EAF9F0

Applied codes, saved settings, completed tasks.

Error--halo-color-status-danger-surface#FFF1F1

Validation and blocking messages.
